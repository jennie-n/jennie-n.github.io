Product Analysis Lab

Engineering Internship at ON Semiconductor
October - December 2020

Summary

At ON Semiconductor, I worked as a product analysis lab engineering intern with the Failure Analysis team. Recently, the team was a part of Quantenna, a wifi start-up that was acquired by ON Semiconductor. This was my first hardware-focused role, so I was able to learn a lot about EE, wifi products, and the quality analysis process.

Scripting Project

My main project for the season was to develop a way to automate the production of TXT files readable by the LitePoint testing machine from the wifi product's data sheet. The code is usable on new wifi products in the future and will be used to reduce the time spent manually entering in hundreds of lines of data into a testing machine.

Technologies Used

I used Microsoft Excel, Visual Basic for Applications (VBA), and LitePoint technology to test wifi broadband and RF chips.

Moving Forward

The Failure Analysis team at ON Semiconductor will be able to use the code and research from this fall to more easily and quickly test wifi products in the lab.
